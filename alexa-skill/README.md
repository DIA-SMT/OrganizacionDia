# Alexa Skill

- `interaction-model-es.json`: modelo para Alexa Developer Console.
- `lambda/index.mjs`: handler para AWS Lambda.
- `lambda/package.json`: metadatos del código de Lambda.
- `hosted/index.js`: handler para Alexa-Hosted, sin cuenta AWS personal.
- `hosted/package.json`: dependencias para Alexa-Hosted.

La guía completa está en `docs/alexa-setup.md`.
